# 手牌記法

mj-tiles で使用できる手牌記法の詳細です。

## 基本記法

### 数牌

- `123m` - 萬子（マンズ）
- `456p` - 筒子（ピンズ）
- `789s` - 索子（ソーズ）

### 字牌

2つの記法をサポート：

- **漢字形式**: `東南西北白發中`
- **z形式**: `1234567z`（東=1z, 南=2z, 西=3z, 北=4z, 白=5z, 發=6z, 中=7z）

### 赤ドラ

2つの記法をサポート：

- **0形式**: `0m`, `0p`, `0s`
- **r5形式**: `r5m`, `r5p`, `r5s`

## 使用例

```tsx
// 数牌のみ
<Tiles hand="123m456p789s" />

// 字牌（漢字）
<Tiles hand="東南西北白發中" />

// 字牌（z形式）
<Tiles hand="1234567z" />

// 国士無双（z形式）
<Tiles hand="19m19p19s1234567z" />

// 赤ドラ（0形式）
<Tiles hand="1230m456p789s" />

// 赤ドラ（r5形式）
<Tiles hand="123mr5p789s" />
<Tiles hand="r5mr5pr5s" />  // 赤5萬、赤5筒、赤5索
```

## 副露記法（拡張）

副露（鳴き）の表現には `parseHandExtended` を使用します。

```tsx
import { parseHandExtended } from "mj-tiles/core";

const hand = parseHandExtended("123m 1-23p 55=5s 東南西");
// → 門前牌 + チー + ポン + 字牌
```

### 新篠ゆう方式（推奨）

記号で鳴きの種類と位置を表現：

| 記法 | 説明 | 例 |
|------|------|-----|
| `-` | チー | `1-23m`（左鳴き）、`12-3m`（右鳴き） |
| `=` | ポン | `5=55p`（左鳴き）、`55=5p`（右鳴き） |
| `=` (4枚) | 大明槓 | `1=111s` |
| `+` | 加槓 | `+1111s` |
| `o...o` | 暗槓 | `o1111so` |

### 牌画作成くん方式

プレフィックスで牌の状態を表現：

| プレフィックス | 説明 | 例 |
|--------------|------|-----|
| `y` | 横向き牌 | `y5m` |
| `o` | 伏せ牌 | `o5m` |
| `a5` | 赤ドラ | `a5m`（= `0m`） |

### 字牌の独自記号（牌画作成くん方式）

| 記号 | 牌 |
|------|-----|
| `t` | 東 |
| `n` | 南 |
| `s` | 西 |
| `p` | 北 |
| `h` | 白 |
| `r` | 發 |
| `c` | 中 |
